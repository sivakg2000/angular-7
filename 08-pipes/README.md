# Pipes

| Type   | pipe      | Examples    | Syntax                 |
| ------ | --------- | ----------- | ---------------------- |
| String | None      | Test system | `{{name}}`             |
|        | lowercase | test system | `{{name | lowercase}}` |
|        | uppercase | TEST SYSTEM | `{{name | uppercase}}` |
|        | titlecase | Test System | `{{name | titlecase}}` |
